## 목적
1. Eclipse를 사용하지 않고 STS를 직접 다운받아 사용합니다.
2. 공부했던 Spring의 기본 개념을 숙지하고 적용하면서 복습
3. 리액터, WebFlux, 배포 등 Spring의 심화 개념 공부
4. 제작한 프로젝트를 실제 배포해봅니다.
